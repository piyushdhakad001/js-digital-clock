# Digital Clock 🕐

A simple digital clock built with vanilla JavaScript.
Displays real-time hours, minutes, and seconds with AM/PM format.

---

## Features
- 🕐 Shows real-time current time
- 🔄 Updates every second automatically
- 🌙 AM/PM format

---

## Tech Used
- HTML
- CSS
- JavaScript (Vanilla)

---

## How to Run
1. Clone the repo
2. Open `index.html` in your browser
